# hospital_registration_system
Python+Django+MySQL+Redis的医院挂号系统

1.0版本功能：
患者：登录、注册、挂号（选择科室、选择医生、选择时间、填写病情、支付宝支付）、展示挂号单。
医生：登录、展示并处理挂号单。
管理员：增删查改数据。
